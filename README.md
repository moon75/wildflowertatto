# Wildflower Tattoo

Next.js 14 (App Router) + Tailwind CSS. Sanity will be added later once the site is shaped.

## Setup

```bash
npm install
npm run dev
```

App: http://localhost:3000

## Routes

Main nav:
- `/` Home
- `/about` About artist
- `/portfolio` Portfolio
- `/services` Services
- `/contact` Contact

Info nav:
- `/preparation-tips`
- `/aftercare`
- `/day-schedule`
- `/terms-deposit`
